# Usage
<!-- usage -->
```sh-session
$ npm install -g auto-browser
$ auto-browser COMMAND
running command...
$ auto-browser (-v|--version|version)
auto-browser/0.0.0 darwin-x64 node-v13.1.0
$ auto-browser --help [COMMAND]
USAGE
  $ auto-browser COMMAND
...
```
